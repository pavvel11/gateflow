How Element Protection Works

With Sellf element protection, you can:

  • 🎯 Protect specific elements
  • 🔓 Keep other content public
  • 🎨 Customize protection behavior
  • 📊 Track engagement per element
// Element protection with data-sellf-product attribute // Just use data-sellf-product attribute on elements <div data-sellf-product="pro-toolkit"> Content only for pro-toolkit users </div> <div data-sellf-product="vip-masterclass"> Content only for vip-masterclass users </div> // Fallback content shown when access is denied <div data-no-access="true"> This content is shown when user doesn't have access </div> // Script usage for element protection: // <script src="/api/sellf"></script>

Protection Modes

Different ways to handle protected content:

  • hide - Hide element completely
  • blur - Blur content with overlay
  • lock - Show lock icon
  • custom - Custom replacement
// Protection modes: // - Elements with data-sellf-product are protected // - Elements with data-no-access show when access denied // - Just add the script and use data attributes
